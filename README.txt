TechBot

Some humble 2d game coded in lua with the löve 2d engine.
You are a cyborg on a tray trying to survive a plasma rain.
left:	to go left
right:	to go right
down:	to split
up:	to jump

Lua 5.3.2 and löve 2d required, place all the file in a directory and use the command "love directory_name"
